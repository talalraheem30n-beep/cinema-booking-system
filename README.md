# 🎬 Cinema Booking System (C++ OOP)

## 📌 Description
This project is a console-based Cinema Booking System developed in C++ using Object-Oriented Programming concepts.  
It allows users to view movies, select seats, process payments, and manage bookings through a structured command-line interface.

The main focus of this project is to demonstrate clean OOP design and logical problem-solving rather than graphical interface development.

---

## ⚙️ Features
- View available movies with ticket prices
- Display seat layout (5x5 grid system)
- Book multiple seats (with validation)
- Prevent duplicate or already booked seat selection
- Payment system (Cash / Card simulation)
- Optional 10% discount feature
- Total revenue tracking

---

## 🧠 OOP Concepts Used
- **Abstraction** → Base class `Item`
- **Inheritance** → `Movie` class derived from `Item`
- **Encapsulation** → Private data with getters/setters
- **Polymorphism** → Virtual function `showInfo()`
- **Dynamic Memory Management** → Use of pointers and destructors
- **Copy Constructor** → Implemented in `SeatMap`
- **Function Pointer** → Used for discount calculation
